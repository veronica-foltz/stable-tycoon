Stable Tycoon

Stable Tycoon is a horse-care simulation game built with Godot.

Players manage a horse stable by feeding, grooming, training, and caring for their horse while balancing hunger, happiness, energy, health, money, and daily progression.

Features

* Horse care gameplay loop
* Hunger, happiness, energy, and health systems
* Animated horse sprite
* Day progression system
* Random daily events
* In-game store screen
* Timer-based stat decay
* Web playable build
* Mobile browser support

Tech Used

* Godot Engine
* GDScript
* HTML5
* JavaScript
* Krita
* Git & GitHub
* itch.io (web deployment)

Gameplay

Players must keep their horse healthy and happy by:

* Feeding
* Resting
* Grooming
* Training

Neglecting stats can cause the horse to become sick.
